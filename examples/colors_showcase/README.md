# Colors Showcase

This example demonstrates the specialized color palette and semantic colors added to the Refast framework.

## Features

- **Semantic Colors**: Success, Warning, Info, Destructive (Failure)
- **Extended Palette**: 50-950 shades for Red, Orange, Yellow, Green, Teal, Blue, Purple, Pink, Gray
- **Component Usage**: Examples of Alerts, Buttons, and Badges using these colors.

## Running the Example

Run the app using the python interpreter:

```bash
python examples/colors_showcase/app.py
```

Then visit http://localhost:8000 in your browser.
