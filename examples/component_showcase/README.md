# Component Showcase Example

This example demonstrates all the Stage 9 Radix UI components available in Refast.

## Features Demonstrated

### Form Controls
- **Switch** - Toggle switches with state management
- **Slider** - Range sliders with real-time value updates
- **Toggle/ToggleGroup** - Toggle buttons for text formatting
- **Combobox** - Searchable dropdown selection

### Navigation
- **Breadcrumb** - Navigation breadcrumbs
- **Tabs** - Tabbed content sections
- **Pagination** - Page navigation controls
- **Menubar** - Application menu bar

### Overlays
- **AlertDialog** - Confirmation dialogs with actions
- **Sheet** - Slide-out panels for forms
- **Popover** - Popup menus and quick actions
- **HoverCard** - Hover-triggered info cards

### Utility Components
- **Separator** - Visual dividers
- **Collapsible** - Expandable/collapsible sections
- **ScrollArea** - Custom scrollable containers
- **AspectRatio** - Fixed aspect ratio containers

## Running

```bash
cd examples/component_showcase
uvicorn app:app --reload
```

Then open http://localhost:8000 in your browser.
