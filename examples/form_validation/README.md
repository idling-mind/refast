# Form Validation Example

This example demonstrates how to build a registration form with validation using Refast.

## Features Demonstrated

- Form handling with `Input` components
- State management for form fields
- Validation logic in Python
- Error and Success feedback using `Alert`

## Running the Example

1. Ensure you have the environment set up.
2. Run the app:

```bash
cd examples/form_validation
python app.py
```

3. Open http://localhost:8000 in your browser.
