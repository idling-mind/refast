# SPA Navigation Example

This example demonstrates how to build a Single Page Application (SPA) with navigation using Refast.

## Features Demonstrated

- SPA navigation (switching content without page reload)
- Conditional rendering based on state
- Shared layout components (Navbar)
- State persistence across "pages"

## Running the Example

1. Ensure you have the environment set up.
2. Run the app:

```bash
cd examples/multi_page
python app.py
```

3. Open http://localhost:8000 in your browser.
