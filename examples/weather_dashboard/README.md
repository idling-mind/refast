# Weather Dashboard Example

This example demonstrates how to build a dynamic dashboard with Refast.

## Features Demonstrated

- Data display with `Card` and `Badge` components
- Grid layout using Tailwind CSS classes
- Dynamic content updates via callbacks
- State management for data

## Running the Example

1. Ensure you have the environment set up.
2. Run the app:

```bash
cd examples/weather_dashboard
python app.py
```

3. Open http://localhost:8000 in your browser.
